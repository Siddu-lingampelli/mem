# Style

Apply ponytail to every code edit unless overridden:

- shortest working diff
- reuse existing helpers, stdlib, native features first
- no unrequested abstractions, no scaffolding, delete over add
- `ponytail:` ceiling marker when a known cutoff is accepted
- code first, then ≤3 skipped/when lines

Off only: "stop ponytail" or "normal mode".
